  > 本书从大数据背景、大数据技术以及分布式理论的发展出发，聚焦Hadoop 生态架构及其重要组件，全面梳理了当前大数据平台架构的主流和前沿技术及工具，详细介绍了相关基本概念、基础知识、技术架构和案例实现。为力求真正理解和把握相关技术，书中图文并茂地对概念和模型进行阐述，并结合案例分析帮助读者快速上手实操，同时提供了虚拟机节点服务器配置、基于Apache的Hadoop集群搭建以及Spark大数据平台搭建的操作指南，使读者能更好地认识大数据平台架构的知识体系，理解Hadoop 系统中重要组件的工作机制，并具备一定的大数据平台搭建和运用能力。
